
# Data Analytics Project Using Python, MySQL, and Power BI

## Project Overview

This project demonstrates an end-to-end data analytics workflow, covering data extraction, cleaning, exploration, querying, visualization, and reporting. The objective is to transform raw data into meaningful business insights using industry-standard analytics tools.

The project integrates Python for data preprocessing and analysis, MySQL for querying and data management, and Power BI for creating interactive dashboards. A presentation report was developed to effectively communicate key findings and recommendations.

---

## Objectives

* Perform data cleaning and preprocessing.
* Conduct Exploratory Data Analysis (EDA) to identify trends and patterns.
* Extract business insights using SQL queries.
* Develop interactive dashboards and visual reports.
* Present findings in a professional format for stakeholders.

---

## Project Workflow

### 1. Data Loading

* Imported dataset into Python environment.
* Performed initial inspection of data structure and quality.

### 2. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data formats.
* Converted columns to appropriate data types.

### 3. Exploratory Data Analysis (EDA)

* Generated descriptive statistics.
* Identified trends and correlations.
* Visualized distributions and patterns.
* Detected outliers and anomalies.

### 4. SQL Analysis

* Loaded cleaned data into MySQL.
* Executed analytical SQL queries.
* Generated business insights using aggregation, filtering, joins, and ranking functions.

### 5. Dashboard Development

Created an interactive Power BI dashboard featuring:

* KPI Cards
* Category Performance
* Customer Behavior Analysis
* Interactive Filters and Slicers

### 6. Reporting

* Prepared a comprehensive analytical report.
* Summarized findings and recommendations.
* Created presentation slides for stakeholder communication.
